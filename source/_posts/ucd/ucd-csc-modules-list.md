## Autumn Trimester

1. **Programming I (Conversion)**
   - **内容**：该课程是编程的入门课程，主要学习编程的基础概念，如变量、循环、条件判断、函数等。
   - 推荐视频：
     - YouTube: “[Python Programming for Beginners](https://www.youtube.com/watch?v=_uQrJ0TkZlc)” by Corey Schafer
     - YouTube: [Programming with Mosh - Python Programming Course](https://www.youtube.com/watch?v=_uQrJ0TkZlc)
2. **Relational Databases & Information Systems (Conversion)**
   - **内容**：学习关系数据库的基本概念，包括SQL查询、数据建模、数据库设计、索引等。
   - 推荐视频：
     - YouTube: “[Database Design Course](https://www.youtube.com/watch?v=ztHopE5Wnpc)” by Caleb Curry
     - YouTube: [freeCodeCamp.org - SQL Tutorial](https://www.youtube.com/watch?v=HXV3zeQKqGY)
3. **Python OOP (Conv)**
   - **内容**：深入了解Python中的面向对象编程，包括类、对象、继承和多态。
   - 推荐视频：
     - YouTube: “[Python OOP Tutorial](https://www.youtube.com/watch?v=JeznW_7DlB0)” by Corey Schafer
     - YouTube: [Tech With Tim - Python Object Oriented Programming](https://www.youtube.com/watch?v=JeznW_7DlB0)
4. **Operating Systems (Conversion)**
   - **内容**：学习操作系统的核心概念，如进程管理、内存管理、文件系统、并发等。
   - 推荐视频：
     - YouTube: “[Operating Systems](https://www.youtube.com/watch?v=26QPDBe-NB8)” by CrashCourse
     - YouTube: [Neso Academy - Operating System Lectures](https://www.youtube.com/playlist?list=PLBlnK6fEyqRhX6r2uhhlubuF5QextdCSM)
5. **Web Application Development (Conversion)**
   - **内容**：主要学习网页应用的开发基础，涉及前端和后端的基本技术如HTML、CSS、JavaScript、数据库等。
   - 推荐视频：
     - YouTube: “[The Web Developer Bootcamp](https://www.youtube.com/watch?v=zJSY8tbf_ys)” by Colt Steele
     - YouTube: [Traversy Media - Full Stack Web Development Course](https://www.youtube.com/watch?v=APZ4c7FNK0Y)
6. **Computational Thinking (Conversion)**
   - **内容**：计算思维训练课程，涵盖问题分解、抽象、算法设计等基本技能。
   - 推荐视频：
     - YouTube: “[Computational Thinking - An Introduction](https://www.youtube.com/watch?v=gM0d3T8Q4Ec)” by Harvard CS50
     - YouTube: [Harvard CS50 - Introduction to Computer Science](https://www.youtube.com/watch?v=8mAITcNt710)

## Spring Trimester

1. **Data Structures & Algorithms (Conversion)**
   - **内容**：学习数据结构和算法的基础知识，包括数组、链表、栈、队列、排序和查找等。
   - 推荐视频：
     - YouTube: “[Data Structures Easy to Advanced Course](https://www.youtube.com/watch?v=RBSGKlAvoiM)” by WilliamFiset
     - YouTube: [freeCodeCamp.org - Data Structures and Algorithms](https://www.youtube.com/watch?v=8hly31xKli0)
2. **Networks & Internet Systems (Conversion)**
   - **内容**：主要讲授网络通信的基础知识，如OSI模型、TCP/IP协议、路由等。
   - 推荐视频：
     - YouTube: “[Computer Networking Full Course](https://www.youtube.com/watch?v=qiQR5rTSshw)” by Joey Chan
     - YouTube: [NetworkChuck - Networking Basics](https://www.youtube.com/watch?v=qiQR5rTSshw)
3. **Computer Architecture & Organisation (Conversion)**
   - **内容**：学习计算机硬件结构，包括CPU、存储、指令集、体系结构等。
   - 推荐视频：
     - YouTube: “[Computer Organization and Architecture](https://www.youtube.com/watch?v=vJwcGGtO9J4)” by Neso Academy
     - YouTube: [CrashCourse - Computer Science: Hardware](https://www.youtube.com/playlist?list=PL8dPuuaLjXtP7Kebh6HEZzkJbhRJ7eZ3c)
4. **Java Programming (Conv)**
   - **内容**：Java编程基础，包括语法、类、接口、异常处理、集合框架等。
   - 推荐视频：
     - YouTube: “[Java Programming - Full Course](https://www.youtube.com/watch?v=RRubcjpTkks)” by BroCode
     - YouTube: [Bro Code - Java Full Course](https://www.youtube.com/watch?v=xk4_1vDrzzo)
5. **Software Engineering (Conv)**
   - **内容**：学习软件工程的基础理论和实践，包括软件开发生命周期、需求分析、设计模式等。
   - 推荐视频：
     - YouTube: “[Software Engineering Full Course](https://www.youtube.com/watch?v=ujbE0mzX-CU)” by Kunal Kushwaha
     - YouTube: [Fireship - Software Engineering](https://www.youtube.com/watch?v=ZtNs30L72CY)
6. **Data Analytics (Conversion)**
   - **内容**：数据分析的基础课程，学习数据处理、统计分析和数据可视化等内容。
   - 推荐视频：
     - YouTube: “[Data Analytics Full Course](https://www.youtube.com/watch?v=6aF2GrRzJWg)” by Simplilearn
     - YouTube: [freeCodeCamp.org - Data Analysis with Python](https://www.youtube.com/watch?v=r-uOLxNrNk8)
