# Fundamental APIs

## Java

### Stack

#### Stack Class

:star:继承关系

```
Collections(I) -> List(I) -> Vector(C) -> Stack(C)
```

:star:线程安全

Vector 是线程安全的, Stack 也是线程安全的, 故写算法题时性能较差

:star:主要方法

```java
boolean empty();
boolean isEmpty();

E peek(); // throws EmptyStackException when empty
E pop(); // throws EmptyStackException when empty
E push(E e);
int search(Object o); // Returns the 1-based position where an object is on this stack.
```

#### Deque Interface & List Interface

更加推荐使用 Deque 或 List 来实现栈

### Queue

#### Queue Interface

:star:继承关系

```
Collection(I) -> Queue(I)
```

:star:主要方法

```java
boolean add(E e); // throwing an IllegalStateException if no space is currently available
E remove(E e); // throws NoSuchElementException when empty
E element(); // throws NoSuchElementException when empty

boolean offer(E e);
E poll(E e); // return null when empty
E peek(E e); // return null when empty
```

:star:主要实现类

| 实现类              | 描述                                                 | 特点                                         |
| ------------------- | ---------------------------------------------------- | -------------------------------------------- |
| **`LinkedList`**    | 基于链表的双端队列实现，既可以用作队列也可以用作栈。 | 非线程安全，性能较好                         |
| **`ArrayDeque`**    | 基于动态数组的双端队列实现。                         | 非线程安全，性能更优于 `LinkedList`          |
| **`PriorityQueue`** | 基于堆的优先级队列。                                 | 非线程安全，元素按自然顺序或自定义比较器排序 |

#### Deque Interface

:star:继承关系

```
Collection(I) -> Queue(I) -> Deque(I)
```

:star:主要方法

|             | **First Element (Head)** | **Last Element (Tail)**     |                      |                           |
| ----------- | ------------------------ | --------------------------- | -------------------- | ------------------------- |
|             | *Throws exception*       | *Special value*             | *Throws exception*   | *Special value*           |
| **Insert**  | `void addFirst(E e);`    | ` boolean offerFirst(E e);` | `void addLast(E e);` | `boolean offerLast(E e);` |
| **Remove**  | `E removeFirst();`       | `E pollFirst();`            | `E removeLast();`    | `E pollLast();`           |
| **Examine** | `E getFirst();`          | `E peekFirst();`            | `E getLast();`       | `E peekLast();`           |



| **`Queue` Method** | **Equivalent `Deque` Method** |
| ------------------ | ----------------------------- |
| `add(e)`           | `addLast(e)`                  |
| `offer(e)`         | `offerLast(e)`                |
| `remove()`         | `removeFirst()`               |
| `poll()`           | `pollFirst()`                 |
| `element()`        | `getFirst()`                  |
| `peek()`           | `peekFirst()`                 |

:star:主要实现类

`ArrayDeque`, `LinkedList`

## [232. Implement Queue using Stacks](https://leetcode.com/problems/implement-queue-using-stacks/)

#### Java

```java
class MyQueue {
    private Stack<Integer> stk0;
    private Stack<Integer> stk1;

    public MyQueue() {
        this.stk0 = new Stack<>();
        this.stk1 = new Stack<>();
    }
    
    public void push(int x) {
        stk0.push(x);
    }
    
    public int pop() {
        this.peek();
        return stk1.pop();
    }
    
    public int peek() {
        if ( !stk1.isEmpty() ) return stk1.peek();
        while ( !stk0.isEmpty() ) {
            stk1.push(stk0.pop());
        }
        return stk1.peek();
    }
    
    public boolean empty() {
        return stk0.isEmpty() && stk1.isEmpty();
    }
}
```

`stack` 是 RAII（Resource Acquisition Is Initialization）类型的对象，生命周期由其作用域自动管理，它的析构函数会自动释放内部的内存资源。

只要类中没有**动态内存分配**（例如 `new` 或 `malloc`），则不需要手动编写析构函数。

```rust
    sz: usize,
    stk0: Vec<i32>,
    stk1: Vec<i32>,
}
/** 
 * `&self` means the method takes an immutable reference.
 * If you need a mutable reference, change it to `&mut self` instead.
 */
impl MyQueue {

    fn new() -> Self {
        MyQueue {
            sz: 0,
            stk0: Vec::new(),
            stk1: Vec::new(),
        }
    }
    
    fn push(&mut self, x: i32) {
        self.stk0.push(x);
        self.sz += 1;
    }
    
    fn pop(&mut self) -> i32 {
        if self.sz == 0 {
            return -1;
        }
        self.sz -= 1;
        if self.stk1.is_empty() {
            while !self.stk0.is_empty() {
                self.stk1.push(self.stk0.pop().unwrap());
            }
        }
        self.stk1.pop().unwrap()
    }
    
    fn peek(&mut self) -> i32 {
        if self.stk1.is_empty() {
            while !self.stk0.is_empty() {
                self.stk1.push(self.stk0.pop().unwrap());
            }
        }
        self.stk1[self.stk1.len() - 1]
    }
    
    fn empty(&self) -> bool {
        self.sz == 0
    }
}
```

## [225. Implement Stack using Queues](https://leetcode.com/problems/implement-stack-using-queues/)

```java
class MyStack {
    private Queue<Integer> q;

    public MyStack() { q = new LinkedList<>(); }
    
    public void push(int x) { q.offer(x); }
    
    public int pop() {
        int sz = q.size();
        for (int i = 0; i < sz - 1; i ++) {
            int tmp = q.poll();
            q.offer(tmp);
        }
        return q.poll();
    }
    
    public int top() {
        int sz = q.size();
        int tmp;
        for (int i = 0; i < sz - 1; i ++) {
            tmp = q.poll();
            q.offer(tmp);
        }
        tmp = q.poll();
        q.offer(tmp);
        return tmp;
    }
    
    public boolean empty() { return q.isEmpty(); }
}
```

## [20. Valid Parentheses](https://leetcode.com/problems/valid-parentheses/)

#### Java

```java
class Solution {
    public boolean isValid(String s) {
        Stack<Character> stk = new Stack<>();
        for (char ch : s.toCharArray()) {
            if (stk.isEmpty()) stk.push(ch);
            else {
                char peek = stk.peek();
                if (peek == '(' && ch == ')') stk.pop();
                else if (peek == '[' && ch == ']') stk.pop();
                else if (peek == '{' && ch == '}') stk.pop();
                else stk.push(ch);
            }
        }
        return stk.isEmpty();
    }
}
```

```java
class Solution {
    public int evalRPN(String[] tokens) {
        Stack<Integer> stk = new Stack<>();
        for ( String str : tokens ) {
            try {
                int num = Integer.parseInt(str);
                stk.push(num);
            } catch (NumberFormatException e) {  // 当字符不是数字走 catch
                int num2 = stk.pop();
                int num1 = stk.pop();
                int res = 0;
                if (str.equals("+")) res = num1 + num2;
                if (str.equals("-")) res = num1 - num2;
                if (str.equals("*")) res = num1 * num2;
                if (str.equals("/")) res = num1 / num2;
                stk.push(res);
            }
        }
        return stk.pop();
    }
}
```

## [239. Sliding Window Maximum](https://leetcode.com/problems/sliding-window-maximum/)

*Monotonic Queue*

#### Java

```java
class Solution {
    private class MonoQueue {
        LinkedList<Integer> q;
        int sz;
        MonoQueue() { q = new LinkedList<>(); }

        void push(int val) {
            sz ++;
            while (!q.isEmpty() && q.getLast() < val) { q.removeLast(); }
            q.addLast(val);
        }

        void remove(int val) {
            if (sz == 0) return;
            sz --;
            if (q.getFirst() == val) q.removeFirst();
        }

        int peek() { return q.getFirst(); }
    }

    public int[] maxSlidingWindow(int[] nums, int k) {
        int[] ans = new int[nums.length - k + 1];
        int idx = 0;
        MonoQueue mq = new MonoQueue();
        for (int i = 0; i < nums.length; i ++) {
            mq.push(nums[i]);
            if (mq.sz == k) {
                ans[idx] = mq.peek();
                idx ++;
                mq.remove(nums[i - k + 1]);
            }
        }
        return ans;
    }
}
```

```java
class Solution {
    public int[] maxSlidingWindow(int[] nums, int k) {
        int[] ans = new int[nums.length - k + 1];
        int idx = 0;
        Deque<Integer> dq = new ArrayDeque<>();
        for (int i = 0; i < nums.length; i ++) {
            // insert
            while (!dq.isEmpty() && dq.peekLast() < nums[i]) dq.pollLast();
            dq.offerLast(nums[i]);
            if (i >= k - 1) {
                ans[idx] = dq.peekFirst(); // get max
                // remove
                if (nums[i - k + 1] == dq.peekFirst()) dq.pollFirst();
                idx ++;
            }
        }
        return ans;
    }
}
```

```rust
use::std::collections::VecDeque;
impl Solution {
    pub fn max_sliding_window(nums: Vec<i32>, k: i32) -> Vec<i32> {
        let k = k as usize;
        let mut ans = vec![0; nums.len() - k + 1];
        let mut idx: usize = 0;
        let mut mq = VecDeque::with_capacity(k);
        for (i, &num) in nums.iter().enumerate() {
            while let Some(back) = mq.back() {
                if *back < num { mq.pop_back(); }
                else { break; } // 别漏了
            }
            mq.push_back(num);
            if i >= k - 1 {
                if let Some(front) = mq.front() {
                    ans[idx] = *front;
                    idx += 1;
                }
                if nums[i - k + 1] == ans[idx - 1] {
                    mq.pop_front();
                }
            }
        }
        ans
    }
}
```

## [347. Top K Frequent Elements](https://leetcode.com/problems/top-k-frequent-elements/)

*Heap* [215. Kth Largest Element in an Array](https://leetcode.com/problems/kth-largest-element-in-an-array/) 手写小顶堆

#### Java

```java
class Solution {
    public int[] topKFrequent(int[] nums, int k) {
        Map<Integer, Integer> freq = new HashMap<>();
        for (int num : nums) freq.put(num, freq.getOrDefault(num, 0) + 1);
        Queue<Integer> heap = new PriorityQueue<>((e1, e2) -> freq.get(e1) - freq.get(e2));
        for (int key : freq.keySet()) {
            heap.offer(key);
            if (heap.size() > k) heap.poll();
        }
        return heap.stream().mapToInt(Integer::intValue).toArray();
    }
}
```

#### C++

```c++
class Solution {
    class MyComparator {
    public: bool operator()(const pair<int, int>& p1, const pair<int, int>& p2) { return p1.second > p2.second; }
    };
public:
    vector<int> topKFrequent(vector<int>& nums, int k) {
        unordered_map<int, int> freq;
        for (int num : nums) freq[num] ++;
        priority_queue<pair<int, int>, vector<pair<int, int>>, MyComparator> pq;
        for (unordered_map<int, int>::iterator it = freq.begin(); it != freq.end(); it ++) {
            pq.push(*it);
            if (pq.size() > k) pq.pop();
        }
        vector<int> ans(k);
        for (int i = k - 1; i > -1; i --) {
            ans[i] = pq.top().first;
            pq.pop();
        }
        return ans;
    }
};
```

```rust
use std::collections::{BinaryHeap, HashMap};
impl Solution {
    pub fn top_k_frequent(nums: Vec<i32>, k: i32) -> Vec<i32> {
        let k = k as usize;
        let mut hash: HashMap<i32, i32> = HashMap::new();
        let mut heap: BinaryHeap<(i32, i32)> = BinaryHeap::with_capacity(k);
        nums.into_iter().for_each(|k| { *hash.entry(k).or_insert(0) -= 1; }); // BinaryHeap 默认是大顶堆, 故减一
        hash.into_iter().for_each(|entry| {
            heap.push((entry.1, entry.0));
            if heap.len() > k { heap.pop(); }
        });
        heap.into_iter().map(|x| x.1).collect::<Vec<i32>>()
    }
}
```
